# SmartGuard-WiFi-Motion-Detector-and-Server

SmartGuard is an IoT-based motion detection system that uses an ESP32 to detect motion through Wi-Fi signal analysis and an optional PIR sensor for stronger motion detection. The ESP32 communicates with a Python Flask server, enabling real-time monitoring and integration with other applications.

## Features

- 📶 Wi-Fi-based motion detection
- 🚶 Optional PIR motion sensor support
- ⚡ Real-time communication with Flask server
- 📡 ESP32 device monitoring
- 🔄 REST API communication
- 🛠️ Arduino Framework (C++)
- 🐍 Python Flask backend

## Installation

### ESP32 Firmware

1. Install Arduino IDE or PlatformIO.
2. Install the ESP32 Board Package.
3. Install required libraries.
4. Update:
   - Wi-Fi SSID
   - Wi-Fi Password
   - Flask Server URL
5. Upload the firmware to ESP32.

---

### Flask Server

Clone the repository.

```bash
git clone https://github.com/your-username/SmartGuard-WiFi-Motion-Detector-and-Server.git
```

Move into the server directory.

```bash
cd Flask-Server
```

Install dependencies.

```bash
pip install -r requirements.txt
```

Run the server.

```bash
python app.py
```

---

## How It Works

1. ESP32 connects to Wi-Fi.
2. Device continuously monitors Wi-Fi signal characteristics.
3. Optional PIR sensor detects strong human movement.
4. Motion events are sent to the Flask server.
5. Flask processes and stores the event.
6. External applications can consume the REST APIs.

---

## REST API

| Method | Endpoint | Description |
|---------|----------|-------------|
| POST | `/motion` | Receive motion event |
| GET | `/status` | Device status |
| GET | `/stream` | Camera/Stream URL (if enabled) |
